# Auto Language  for VS Code

 <p>Twitter: <a href="https://x.com/BasheirHassan">@BasheirHassan</a></p>


هذه الإضافة تقوم بتبديل لغة لوحة المفاتيح تلقائياً بناءً على سياق النص الذي تكتب فيه (عربي أو إنجليزي).

## المميزات
- **تبديل تلقائي**: تكتشف اللغة من الأحرف المحيطة بالمؤشر وتقوم بتبديل لغة النظام.
- **دعم اللغة العربية والإنجليزية**: تدعم نطاقات اليونيكود العربية والأحرف اللاتينية.
- **إشعارات**: تنبيهك عند تبديل اللغة (يمكن إيقافها من الإعدادات).
- **شريط الحالة**: عرض حالة الإضافة واللغة المكتشفة في شريط الحالة.

## الإعدادات
يمكنك التحكم في الإضافة من خلال الإعدادات:
- `autolanguage.enabled`: تفعيل أو تعطيل الإضافة.
- `autolanguage.showNotifications`: إظهار الإشعارات.
- `autolanguage.showStatusBar`: إظهار شريط الحالة.
